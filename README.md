## pong-game

### Description
This is a classical pong game implementation. Enjoy :D

### How to run
- First install the dependencies via `npm install`.
- Then, you can run the application with `node index.js` command.
- Game is served at `localhost:4000`

### Used technologies
- client side
  - vanilla javascript
  - html5 canvas
  - socket.io
- server side
  - express.js
  - socket.io
